NijieDownloader
===============

nijie.info downloader

Feature
===============
- Image browsing by Member ID.
- Image browsing by Search Tag.
- Batch Image downloading by Image ID, Member ID, or Search Tag.

Required Library
===============
HTML Agility Pack
log4net

Compiled on Visual Studio 2010

Supported OS
===============
WinXP SP2 and up with .Net 4.0
